# Student Information Management System (SIMS)

A high-performance, vanilla web application designed for academic administration.

## 🚀 Features
- **Modern SaaS UI**: Minimalist design with CSS Grid/Flexbox.
- **Role-Based Simulation**: Admin, Teacher, and Student views.
- **Full CRUD**: Manage students, courses, and grades.
- **Analytics**: Custom-built charts using HTML5 Canvas API (No libraries).
- **Persistence**: Data remains after refresh via LocalStorage engine.
- **Theme Engine**: System-wide Light/Dark mode toggle.

## 🛠️ Technical Stack
- **Structure**: Semantic HTML5
- **Style**: Custom CSS3 (Variables, Animations, Glassmorphism)
- **Logic**: Modular Vanilla JavaScript (ES6+)

## 📂 Installation
1. Clone or extract the folder.
2. Open `index.html` in a browser.
3. *Recommended*: Use VS Code "Live Server" to ensure modular JS loads correctly.
